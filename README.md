# FPLxStockX
About:
Dashboard for FPL players prices, demand and  statistics with stockX vibes

Requirements:
In order to run the project please make sure the following software is installed:
- Node.js Version 10+
- Yarn 1+
  
To run the FE:  
 - cd frontend
 - yarn
 - yarn start
 
 To run the BE:  
 - cd backend
 - yarn
 - yarn start
